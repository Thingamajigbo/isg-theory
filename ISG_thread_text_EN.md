# ISG Theory — Thread Text (English)

────────────────────────────────────
## A. X (Twitter) Thread — 9 posts
────────────────────────────────────

1/
Everyone obsesses over "how do I prompt AI better?"

But the real question is:
"When does AI actually break?"

Here's a framework I put together. I call it the Information–Signal Gap Theory (ISG) 🧵

2/
The core, in one line:

AI doesn't fail because information is missing.
It fails because it never says it's missing.

AI isn't dangerous because it doesn't know — it's dangerous because it fails quietly.

3/
The decisive difference between humans and AI:

Human: doesn't know → stops and asks. ("I don't know this")
AI: doesn't stop. Keeps going with the exact same confidence.

There's no "I don't know" alarm inside the model.
So failure happens in silence.

4/
AI seems to fail in infinite ways, but information can only be missing in 4 places:

1. Input — when the instruction is vague
2. Memory — when it forgets the goal over a long task
3. Self — when it's wrong but confident
4. World — when it acts irreversibly

That's all of them.

5/
Why exactly 4?

Because information has only 4 sources (input, memory, self, world).
A 5th boundary would need a 5th source — there isn't one.

So this isn't a list I picked. It's a deduction.

6/
"But modern models barely hallucinate."

True — but that's only half the story.

As models improve → the information gap shrinks ✅
But → the signal gap (the ability to say "I don't know") does NOT ❌

Smarter models fill wrong blanks more smoothly. The silence deepens.

7/
This is why ISG doesn't die with model versions.

GPT, Claude, whatever comes next —
as long as it smoothly fills wrong blanks, the 4 boundaries don't disappear.
They just get pushed further back.

8/
So the fix is NOT "make the AI smarter."
We can't touch the model's internals.

Sound the missing alarm from the outside:

1. Demand a blank-free spec
2. Re-inject the goal each step
3. Verify outside the AI
4. Human approval before anything irreversible

9/
To sum up:

AI is dangerous not because it's dumb, but because it fails quietly.
Put an alarm on that silence, and AI becomes controllable.

— Information–Signal Gap Theory (ISG)

Refutations welcome. A theory only gets stronger the more it's tested.
